# Projeto: Amigo Secreto

Este é um projeto simples de **Amigo Secreto** implementado com **HTML, CSS e JavaScript**. O objetivo é criar uma aplicação interativa onde é possível adicionar amigos, realizar o sorteio de forma aleatória e exibir os resultados de maneira dinâmica e amigável.

---

## Funcionalidades

- **Adicionar Amigos**: O usuário pode inserir nomes de amigos para uma lista.
- **Sortear Amigo Secreto**: Após adicionar os amigos, o sorteio aleatório pode ser realizado.
- **Exibir Lista de Amigos**: A lista é atualizada em tempo real conforme os nomes são adicionados.
- **Mostrar Resultado do Sorteio**: O resultado do sorteio é exibido de forma clara na interface.

---

## Como Usar

### 1. Adicionar Amigos
- Digite o nome de um amigo no campo de texto.
- Clique no botão **Adicionar**.
- O nome será automaticamente incluído na lista de amigos.

### 2. Sortear Amigo Secreto
- Após adicionar todos os nomes desejados, clique em **Sortear Amigo**.
- O nome sorteado será exibido na tela.

---

## Tecnologias Utilizadas

- **HTML** – Estrutura da página.
- **CSS** – Estilização da interface (personalizável via `style.css`).
- **JavaScript** – Lógica do sorteio, manipulação da interface e interatividade.
